그래도 아직 부족한 점이 더 많아 공부해야할 필요성을 매우 느끼고 있습니다.

문제점 및 해결과정

github에서 왜 메인이 안 보이지?

BrowserRouter로 작업 후 깃허브에 올렸을 때, 메인이 안 뜸 -> HashRouter로 수정하니 해결

이미지가 안 보여...

작업 후 깃허브에 push했을 때, process.env.PUBLIC_URL를 이용한 public 폴더 안 이미지, background-imgae에서 static/media로 설정된 이미지만 안 보임
-> 이미지경로를 github page주소로 변경하니 보임

제품의 장바구니 버튼을 눌러도 정보가 왜 장바구니로 안 갈까?

export function addCart(itemId) { cart[itemId] = SIDIZ[itemId].title;
localStorage.setItem(SIDIZ_KEY, JSON.stringify(cart)); }
이렇게 함수를 만들어놓고는 props에 대한 이해 부족으로 const handleAddWishlist = () => { addCart(item.id); }; 이렇게 적어 작동 x
-> const handleAddWishlist = (id) => { addCart(id); }; props를 다시 받아 해결
