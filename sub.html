문제점 및 해결과정

clearTimeout이 왜 작동하지 않을까?

처음엔 조건식으로 작성하지 않아서 안 되었다고 생각. 조건식으로 다시 작성해도 작동하지 않음. 변수 i를 typing 함수 내 선언 -> 밖에서 선언하니 작동

스크롤에 따라 dots가 파트마다 하나씩 색깔이 변경되어야 하는데 왜 이전 dot은 기본으로 안 돌아갈까?

main dots가 파란색인 상태에서 project 파트에 오면 project dot이 파란색이 되고 main dot이 기본으로 다시 바뀌어야 하는데, 점차적으로 파란색만 늘어남. if조건문에서 forEach를 이용해서 전체 dots를 classList.remove하고 그 외는 idx값에 따라 classList.add를 적용해 해결

마우스 커서가 왜 안 보일까?

마우스 커서를 변경하려고 css와 js를 작성했으나 화면에 보이지 않음. 알고보니 contents에 가려져서 보이지 않았음. -> z-index값을 줘서 해결

슬라이드가 왜 옆으로 안 넘어갈까?

겉으로는 슬라이더의 형태이나 기능이 안 됨 -> slider.addEventListener("mousemove", (e) => { if (sliderGrabbed) { slider.parentElement.scrollLeft -= e.movementX; } }); 이렇게 값을 주니 해결

skillList에서 각각 시간차 애니메이션

스크롤을 내리면 각각 시간차로 스르륵 나오길 바람. 스크롤에 따른 이벤트로 각각 시간을 다르게 줬더니 코드 어지러워보임 -> AOS(Animate On Scroll Library) 를 사용해서 깔끔해보이게 함
